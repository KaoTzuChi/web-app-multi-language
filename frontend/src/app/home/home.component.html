~~ Multiple languages example ~~

  • Available languages: en=English, es=Español, zh-tw=繁體中文, zh-cn=简体中文
  • Default language: en
  • Initial language from browser: {{ this.translate.getBrowserLang() }}
  • Go to product page, and use current language: http://localhost:4200/product
  • Go to product page, and use the language in url: http://localhost:4200/es/product
